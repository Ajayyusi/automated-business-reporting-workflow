# Data

This folder contains synthetic sample data for a UAE SME service business.

The sample data is generated by `src/generate_sample_data.py` and is safe to publish in a portfolio because it does not contain real customer or financial records.

Generated files:

- `transactions.csv`
- `expenses.csv`
- `income.csv`
- `customers.csv`
- `kpi_targets.csv`
- `monthly_summary.csv`

Cleaned outputs are saved in `data/processed/`.
